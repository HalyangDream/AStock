## 一、头肩底形态核心要领

### 1.1 形态结构（四要素）

```
价格
 ↑
 │    ╱╲          ╱╲          ╱╲
 │   ╱  ╲  H1   ╱  ╲  H2   ╱  ╲    ← 颈线（H1-H2 连线）
 │  ╱    ╲      ╱    ╲      ╱
 │ ╱   L1 ╲    ╱  L2  ╲   ╱ L3
 │╱  左肩   ╲  ╱  头部   ╲╱  右肩
 └────────────────────────────────→ 时间
```

| 要素 | 定义 | 关键约束 |
|---|---|---|
| **左肩 (L1)** | 下跌后首次反弹的低点 | 反弹有量 |
| **头部 (L2)** | 二次下跌创新低 | L2 < L1；下跌缩量（价量背离） |
| **右肩 (L3)** | 三次下跌但不破头部低点 | L3 > L2；量能最低（下跌动能衰竭） |
| **颈线** | H1-H2 高点连线 | 可水平或倾斜；右上倾斜更强 |

### 1.2 成交量规律（核心鉴别点）

| 阶段 | 量能特征 | 含义 |
|---|---|---|
| 左肩下跌 → 反弹 | 放量反弹 | 初次抄底资金进入 |
| 头部下跌 | **缩量创新低** | 卖盘衰竭，价量背离 |
| 头部反弹 | 温和放量 | 多头开始试探 |
| 右肩下跌 | **全程最低量** | 空头力量枯竭 |
| 右肩反弹 → 突破颈线 | **显著放量** | 多头主力进场确认 |

**量能递减规律**：左肩量 > 头部量 > 右肩量（下跌阶段），突破时量能反转放大。

### 1.3 买入时机（三个买点）

| 买点 | 位置 | 风险/收益 | 确认度 |
|---|---|---|---|
| **第一买点** | 右肩企稳处（L3 附近） | 风险最大 / 收益最大 | 低 |
| **第二买点** | 放量突破颈线 | 风险中等 / 收益中等 | 高 |
| **第三买点** | 突破后回抽颈线确认 | 风险最小 / 收益最小 | 最高 |

### 1.4 目标价与止损

- **经典目标价** = 颈线价 + (颈线价 - 头部最低价)
- **保守目标价** = 颈线价 × (1 + 保守百分比)
- **止损位** = 头部最低价（跌破头部意味着形态失败）
- **突破后止损** = 颈线价（回落到颈线以下 = 假突破）

### 1.5 形态有效性条件

| 条件 | 说明 |
|---|---|
| **前置下跌趋势** | 必须有明显下跌才构成"底部反转"；横盘中的三低点不算 |
| **对称性** | 左右肩时间跨度大致对称（右肩可略长）；价格对称（右肩可略高于左肩） |
| **颈线斜率** | 水平或右上倾斜 = 更强信号；右下倾斜过大 = 形态较弱 |
| **突破量能** | 突破颈线当日成交量须明显放大（相对近日均量） |
| **形态跨度** | 通常 30-120 个交易日；过短 (<20日) 或过长 (>200日) 信号意义降低 |

| 维度 | 现有 `headShoulder.py` | 课程要领 | 差距 |

## 三、建议新增到 `stock-course-strategy` skill 的头肩底策略

### 3.1 战法定位

在主策略 27 种战法中新增 **第 28 条：头肩底战法**，归入 **C 组（K 线形态战法）**。

### 3.2 策略定义（建议写入 skill）

```
#### 28. 头肩底战法
- **形态**: 左肩(L1) → 头部(L2, 创新低) → 右肩(L3, 不破头)，H1-H2 连线为颈线
- **量能规律**: 下跌阶段量能递减（左肩量 > 头部量 > 右肩量）；突破颈线放量
- **前提**: 前置下跌趋势 + 颈线斜率不过度右下倾斜
- **买点**:
  - P14a: 右肩企稳（L3 不破 L2 + 量能萎缩）→ 激进买点
  - P14b: 放量突破颈线 → 标准买点（推荐）
  - P14c: 突破后回抽颈线确认 → 保守买点
- **卖出/止损**:
  - 跌破头部低点 L2 → 形态失败，无条件止损
  - 突破后回落颈线以下 → 假突破，止损
- **目标价**: 颈线 + (颈线 - L2)
- **出处**: 经典技术分析 + 课程体系
```

### 3.3 对应 P/S 信号扩展

| 编号 | 名称 | 条件 | 层级 |
|---|---|---|---|
| P14a | 头肩底右肩企稳 | 右肩量萎缩 + 不破头 + 前置下跌 | 加分 |
| P14b | 头肩底颈线突破 | 放量突破颈线 + 量能递减确认 | **高优先级** |
